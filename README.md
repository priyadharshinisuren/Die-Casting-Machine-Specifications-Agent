Key Features
6 Materials Supported: Aluminum, Zinc, Magnesium, Copper, Lead, Tin — each with density, melting point, pouring temps & pressure ranges
Real-Time Calculations — results update instantly as you type:
✅ Required Clamping Force (with safety factor)
✅ Recommended Machine Tonnage (from 80T to 4000T)
✅ Fill Ratio with visual gauge
✅ Metal Pressure, Injection Force
✅ Gate Velocity & Plunger Velocity
✅ PQ² Ratio & Max Casting Area
Smart Recommendations — tells you the ideal machine tonnage, utilization %, and fill ratio advice
Material Reference Table — quick lookup for all material properties
Standard Tonnage Chips — highlights the recommended machine automatically
Print-Ready — clean print layout for documentation
Reset Button — clears all inputs instantly
🧪 How to Use
Enter your die-casting specs (casting area, plunger diameter, shot weight, etc.) and click Calculate Results — the app instantly shows all computed values with status indicators (✅ Optimal / ⚠️ Warning / 🔴 Danger).
